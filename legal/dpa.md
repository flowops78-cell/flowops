# Data Processing Addendum (DPA)

## 1. Roles
User = Data Controller  
Platform = Data Processor  

---

## 2. Scope
The platform processes data strictly on user instruction.

---

## 3. Data Types
- identifiers
- activity records
- metadata

---

## 4. Security Measures
- encrypted transport (HTTPS)
- access controls
- role-based permissions

---

## 5. Subprocessors
Infrastructure providers (e.g. hosting, database).

---

## 6. Breach Notification
In case of breach, reasonable effort will be made to notify affected users.

---

## 7. Data Retention
Data is retained until:
- user deletes it
- account is terminated

---

## 8. Liability
The platform is not responsible for how users interpret or use stored data.
