# Privacy Policy

## 1. Data Collected
We collect only data explicitly entered by users, including:
- identifiers (name, contact)
- operational records
- usage metadata

---

## 2. Data Usage
Data is used solely to:
- operate the application
- provide functionality
- improve system performance

---

## 3. No Data Selling
We do not sell or share user data with third parties.

---

## 4. Data Storage
Data is stored securely using third-party infrastructure providers.

We do not guarantee zero risk but apply reasonable safeguards.

---

## 5. User Control
Users may request:
- data deletion
- data export

---

## 6. Responsibility Boundary
Users are responsible for the legality of stored data.

The platform does not validate or enforce user compliance.
